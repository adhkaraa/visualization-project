# Data Visualization Project

## [Supermarket Sales Dashboard](https://drive.google.com/file/d/1sdhlRohxHZUarhzepCcVB6d17n9aU7dP/view)  
This project visualizes supermarket sales data to track revenue, profit, and customer behavior using PowerBI. It highlights top-performing products, payment preferences, and rating insights to support strategic decisions.

## [“Sesibuk Apa Jadi DPR RI?” — Data Analysis of DPR RI Agendas](https://medium.com/@adhkaraa/sesibuk-apa-jadi-dpr-ri-cc0ca113f01f)
Analyzed publicly available DPR RI agenda data (Oct 2024–May 2025) to quantify legislative workload patterns. Collected, cleaned, and visualized schedule data to identify activity frequency, peak days/times, and variations across legislative bodies, demonstrating how open data can reveal institutional work patterns.
